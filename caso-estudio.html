Alke Wallet

Sistema de Gestión de Billetera Digital

Inicio Proyectos

Descripción

Alke Wallet es una aplicación web desarrollada con Django para gestionar billeteras digitales. Permite crear clientes, cuentas y registrar transacciones (depósitos, retiros y transferencias).

Desafío

Desarrollar una aplicación Django completa con modelo de datos relacional, operaciones CRUD funcionando correctamente y una interfaz de administración integrada.

Solución

Se creó una aplicación con 3 modelos principales:

  • Cliente: Información del usuario
  • Cuenta: Cuentas bancarias con saldo
  • Transacción: Registro de movimientos

Con operaciones CRUD completas y validaciones de datos.

Tecnologías Usadas

Django 6.0.4, Python 3.14, SQLite, Bootstrap 5, HTML/CSS

Aprendizajes

  • Diseño de modelos relacionales con Django ORM
  • Implementación de Class-Based Views para CRUD
  • Validaciones en modelos y formularios
  • Uso del admin panel de Django
  • Buenas prácticas y documentación
